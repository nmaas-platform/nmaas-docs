# Introduction to nmaas Application Catalogue

The nmaas Application Catalogue contains documentation for applications that can be deployed through nmaas, together with guidance for users who deploy application instances and maintainers who add new applications to the catalogue.

This section is organized around the following topics:

- [General Application Deployment](general-app-deployment.md) - explains the common deployment workflow in the nmaas Portal, including the initial configuration step and the GitOps-based configuration update process used by selected applications.
- [Supported Applications](application-list.md) - lists applications currently available in the nmaas catalogue, including their application and chart versions.
- [Adding a New Application](new-application.md) - describes how to prepare and publish a Helm chart repository for an application that should be added to the nmaas catalogue.

## Application Deployment Tutorials

The application-specific tutorials complement the general deployment workflow with configuration details, screenshots, and usage notes for individual tools. They are useful when deploying an application for the first time or when checking application-specific requirements before requesting a new instance.

For a broader walkthrough of the nmaas Portal and the user-facing application lifecycle, see the [User Guide](../guides/user-guide.md).
