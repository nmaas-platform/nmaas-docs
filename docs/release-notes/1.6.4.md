# 1.6.4

## General

[nmaas 1.6.4](https://gitlab.software.geant.org/nmaas/nmaas-platform/-/releases/v1.6.4) is a minor update containing 
user interface improvements and a fix related with domain groups management panel.

- Updated sorting options available on the application market view.
- Application addition or removal is now reflected on the list of applications available on the domain group configuration view.
- Added support for a new type of contact form along with a new email template for vLAB requests.
