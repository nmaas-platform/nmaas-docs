# 1.8.1

## General

[nmaas 1.8.1](https://gitlab.software.geant.org/nmaas/nmaas-platform/-/releases/v1.8.1) is a minor release containing 
a number of smaller enhancements and bug fixes.

- Added fields in application deployment specification to indicate the amount of resources consumed by given application instance
- Improved performance by trimming the data model for some of the platform API endpoints
- Addressed a number of bugs on the Portal side
- Fixed handling of OIDC configuration when SSO integration is disabled
