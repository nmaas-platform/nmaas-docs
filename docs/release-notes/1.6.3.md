# 1.6.3

## General

[nmaas 1.6.3](https://gitlab.software.geant.org/nmaas/nmaas-platform/-/releases/v1.6.3) is a minor update containing 
foremost a support for new portal landing pages and improvements to bulk user account creation.

- New landing pages can be now customized to reflect the purpose of a given nmaas deployment.
- Bulk user deployment process was enhanced with an option to indicate that given user account should allow for SSO login.
- New users with accounts created in bulk process will receive customized email messages with instructions upon first login.
- Fix was applied to drop down version selector on manual application instance version upgrade modal.

## Helm chart changes

Additional value passed to the Portal deployment.

A new variable in the Helm values file `portal.properties.landingPageFlavor` was added to indicate a name to be used
to display appropriate landing page in the Portal. The default flavor is set to `VLAB`.
